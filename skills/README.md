# Skills

This folder contains reusable guidance files for Codex work on HYMIND.

## Current Skills

- `hydrogen_research_skill.md`: gather and structure hydrogen industry intelligence.
- `source_validation_skill.md`: check freshness, credibility, and duplication.
- `report_generation_skill.md`: convert findings into executive report language.

## Guidelines

- Keep each skill focused on one task.
- Write skills so they can be reused in later phases.
- Prefer short, concrete instructions over broad prose.
