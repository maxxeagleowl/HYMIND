# Source Validation Skill

## Purpose

Check whether a source item is credible, recent, and useful before it is used in a report.

## Validation Checks

- Is the source recent enough for the topic?
- Is the source directly relevant?
- Is the source reputable or primary where possible?
- Does the item duplicate something already collected?
- Is the claim stated clearly enough to summarize safely?

## Working Rules

- Prefer primary sources when available.
- Flag unsupported claims.
- Remove duplicates before synthesis.
- Track uncertainty when the evidence is weak.

## Validation Output

- Keep
- Drop
- Needs review
- Notes on why
